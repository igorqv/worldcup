🎯 Apresentação Executiva

📱 O que é este projeto?

Imagina que você quer entender tudo sobre a Copa do Mundo... Quem foi campeão? Quantos gols foram marcados? Quem jogou melhor?

🎯 A IDEIA PRINCIPAL Pegamos TODOS os dados sobre Copa do Mundo (desde 1930!) e colocamos em um lugar bonito e fácil de entender. É como um videogame onde você pode explorar histórias da Copa!

Por que criamos este projeto? 🤔

  • Para deixar os dados da Copa bonitos e fáceis de entender
  • Para descobrir padrões interessantes nos números
  • Para mostrar a história da Copa de forma visual
  • Para tomar decisões melhores com base em dados reais

📊 Os Dados que Usamos

Pegamos 3 arquivos CSV do Kaggle (um site com muitos dados para análise):

🏆

world_cup.csv

O quê: Informações sobre cada Copa do Mundo

Exemplos: Qual país foi campeão? Vice-campeão? Quantos times participaram? Quantas partidas foram?

⚽

matches_1930_2022.csv

O quê: Todos os jogos desde 1930

Exemplos: Quem jogou? Qual foi o resultado? Quantos gols cada time fez?

🎖️

fifa_ranking_2022-10-06.csv

O quê: Ranking dos melhores times do mundo (2022)

Exemplos: Qual time é o melhor? Quantos pontos tem cada um?

💡 Resumindo: São como 3 livros diferentes sobre Copa do Mundo. Um fala dos campeões, outro dos jogos, e o terceiro do ranking dos times!

🛠️ As Ferramentas Mágicas que Usamos

Usamos 4 bibliotecas Python (linguagem de programação) para fazer tudo funcionar:

📦 Biblioteca 🎯 Para quê? 🔍 Exemplo
Streamlit Criar a página web interativa e bonita Quando você clica em um botão e vê os dados mudarem
Pandas Ler, organizar e manipular os dados dos CSVs Separar times por confederação, somar gols, etc
Plotly Criar gráficos bonitos e interativos Os gráficos coloridos que você vê na tela
NumPy Fazer contas rápidas com números Calcular médias, somas, porcentagens
🎨 Pense assim: Se fosse um bolo, Pandas seria os ingredientes (dados), NumPy seria o misturador, Plotly seria a cobertura (gráficos), e Streamlit seria a caixa bonita onde você compra o bolo!

⚙️ Como o Código Funciona?

5 Passos Simples:

1️⃣ LER OS DADOS O código abre os 3 arquivos CSV e carrega todos os dados sobre Copa do Mundo
2️⃣ LIMPAR OS DADOS Tira dados ruins ou incompletos (como jogar fora maçãs podres)
3️⃣ CALCULAR COISAS INTERESSANTES Faz contas para saber quantos gols cada time fez, quem venceu mais, etc
4️⃣ CRIAR GRÁFICOS BONITOS Transforma números chatos em figuras coloridas e interessantes
5️⃣ MOSTRAR TUDO DE FORMA BONITA Coloca tudo junto em uma página web que qualquer pessoa pode usar
💻 É como cozinhar: Você pega os ingredientes (dados), prepara tudo (limpa), faz o prato (calcula), decora (gráficos), e serve bonito (Streamlit)!
